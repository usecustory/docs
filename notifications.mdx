Custory includes an in-app notification center plus channel-based delivery preferences.

## Notification center

Open the notification bell to review updates in one place.

The notification center supports:

- Inbox and archive views
- Mark as read and unread
- Archive and restore
- Open the related work directly from the notification

## Delivery channels

Custory supports these notification channels:

- In-app
- Email
- Slack direct message
- Discord direct message

Use in-app for the default stream. Add direct channels for the updates that need faster attention.

## Notification groups

Custory groups notification preferences into:

- **Mentions**
- **Item updates**
- **System**

System includes things like workspace access events, automation problems, integration health, and external task sync updates.

## Event coverage

Notifications can cover events such as:

- Comment mentions
- Item comments
- Assignment changes
- Item status changes
- Invitation and join request updates
- Automation failures
- Invalid automations
- Integration errors or expiry
- External task title or status changes

## Slack and Discord account linking

You can link your personal Slack or Discord identity to your Custory account.

This is separate from workspace-level channel integrations. It lets Custory deliver direct messages to you as an individual, not only to a shared team channel.

Use this for high-signal updates such as mentions, assignments, automation failures, and integration issues.
